## 掌握的题型模式

### 1. 哈希表配对查找

**题目**：两数之和
**核心思路**：边遍历边查找，用哈希表存储已遍历元素
**适用场景**：在数组中找两个数，使其满足某种关系

### 2. 最大子数组和（Kadane 算法）

**题目**：最大子数组和
**核心思路**：
- 动态规划：dp[i] = max(dp[i-1] + nums[i], nums[i])
- 贪心：如果当前和为负数，就重新开始
**适用场景**：求连续子数组的最大和

### 3. 滑动窗口

**题目**：无重复字符的最长子串
**核心思路**：
- 维护一个窗口 [left, right]
- 窗口不满足条件时收缩，满足条件时扩张
**适用场景**：找最长/最短的子串/子数组

## 学习心得

### 理解加深
1. **数组的遍历方式**：
   - 单层遍历：O(n)
   - 双层遍历：O(n²)
   - 双指针：可以在 O(n) 或 O(n log n) 内完成

2. **哈希表的作用**：
   - 快速查找：O(1)
   - 空间换时间
   - 需要注意键的处理（是否存在、是否重复）

3. **动态规划的基本思想**：
   - 定义状态
   - 状态转移
   - 边界条件
   - 空间优化

### 遇到的问题

1. **边界条件处理**：
   - 空数组/空字符串
   - 单元素情况
   - 全负数情况

2. **代码实现细节**：
   - HashMap 的使用（containsKey, get, put）
   - 数组的初始化
   - 循环的边界（left <= right, left < right）

3. **复杂度分析**：
   - 时间复杂度的计算
   - 空间复杂度的计算
   - 如何通过分析选择更优的解法
