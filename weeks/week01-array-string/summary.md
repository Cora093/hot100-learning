## 掌握的题型模式

### 1. 哈希表配对查找

**题目**：两数之和
**核心思路**：边遍历边查找，用哈希表存储已遍历元素
**适用场景**：在数组中找两个数，使其满足某种关系
**学习进度**：⬜ 待学习

### 2. 最大子数组和（Kadane 算法）

**题目**：最大子数组和
**核心思路**：
- 动态规划：dp[i] = max(dp[i-1] + nums[i], nums[i])
- 贪心：如果当前和为负数，就重新开始
**适用场景**：求连续子数组的最大和
**学习进度**：⬜ 待学习

### 3. 滑动窗口

**题目**：无重复字符的最长子串
**核心思路**：
- 维护一个窗口 [left, right]
- 窗口不满足条件时收缩，满足条件时扩张
**适用场景**：找最长/最短的子串/子数组
**学习进度**：⬜ 待学习

## 代码实现与测试

### Java 文件规范

每道题都对应一个 Java 文件，遵循以下规范：
- **类名**：使用帕斯卡命名法（如 TwoSum、MaxSubArray）
- **文件名**：与类名完全一致
- **算法实现**：**只提供方法签名，不包含实现代码**
- **测试用例**：包含在 main 方法中
- **测试覆盖**：常规情况、边界情况、特殊情况

### 测试执行

项目使用 Maven 进行构建和测试：

```bash
# 编译项目
mvn clean compile

# 运行特定类的测试
mvn exec:java -Dexec.mainClass="TwoSum"
mvn exec:java -Dexec.mainClass="MaxSubArray"
mvn exec:java -Dexec.mainClass="LongestSubstring"

# 批量运行所有测试
./run-tests.sh        # Linux/Mac
run-tests.bat         # Windows
```

**注意**：
- 禁止使用 `javac` 直接编译
- 必须通过 Maven 进行编译和运行

## 学习心得

### 理解加深
1. **数组的遍历方式**：
   - 单层遍历：O(n)
   - 双层遍历：O(n²)
   - 双指针：可以在 O(n) 或 O(n log n) 内完成

2. **哈希表的作用**：
   - 快速查找：O(1)
   - 空间换时间
   - 需要注意键的处理（是否存在、是否重复）

3. **动态规划的基本思想**：
   - 定义状态
   - 状态转移
   - 边界条件
   - 空间优化

### 遇到的问题

1. **边界条件处理**：
   - 空数组/空字符串
   - 单元素情况
   - 全负数情况

2. **代码实现细节**：
   - HashMap 的使用（containsKey, get, put）
   - 数组的初始化
   - 循环的边界（left <= right, left < right）

3. **复杂度分析**：
   - 时间复杂度的计算
   - 空间复杂度的计算
   - 如何通过分析选择更优的解法
