**核心技巧**：
- 双指针（同向、相向）
- 原地修改
- 滑动窗口（简单版）

### 字符串处理

字符串本质上是字符数组，处理方式与数组类似，但需要注意一些特殊操作。

**核心技巧**：
- 字符串遍历
- 子串判断
- 字符统计

## 掌握的题型模式

### 1. 哈希表配对查找

**题目**：两数之和
**核心思路**：边遍历边查找，用哈希表存储已遍历元素
**适用场景**：在数组中找两个数，使其满足某种关系
**学习进度**：✅ 已掌握

### 2. 最大子数组和（Kadane 算法）

**题目**：最大子数组和
**核心思路**：
- 动态规划：dp[i] = max(dp[i-1] + nums[i], nums[i])
- 贪心：如果当前和为负数，就重新开始
**适用场景**：求连续子数组的最大和
**学习进度**：⬜ 待学习

### 3. 滑动窗口

**题目**：无重复字符的最长子串
**核心思路**：
- 维护一个窗口 [left, right]
- 窗口不满足条件时收缩，满足条件时扩张
**适用场景**：找最长/最短的子串/子数组
**学习进度**：⬜ 待学习
